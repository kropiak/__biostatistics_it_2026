# Biostatistics, IT part

**Agenda:**

* introduction to Python programming language,
* using pandas and numpy for loading and transforming data,
* using pandas for data analysis,
* data visualization with pandas and python.


**Laboratory configuration:**

During our lectures all examples and exercises will be held using Google Colab through a web browser. Lecturer will prepare Jupyter Notebooks with examples and exercises for online usage and result sharing.

**Course completion**

Students are required to complete the exercises assigned during the course and to share their solutions in accordance with the instructions provided in the description of each laboratory. On this basis, the percentage of completion of individual tasks and the final average will be determined.
